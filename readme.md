# PyOx
PyOx is a Python-based compiler builder inspired by classic tools such as Lex, Yacc, and Ox.
It provides a clean, Pythonic approach to defining lexers, parsers, and semantic actions for building programming languages and domain-specific languages.

## Status
Early development. Interfaces and features are subject to change.
